---

Use the second-opinion workflow for:

"$ARGUMENTS"

Phase 1:
- State the current position clearly.
- List 2-4 key assumptions.
- List the main risks or open questions.
- Produce a portable prompt that can be copied to Claude Code, Ollama, ChatGPT, Gemini, or another LLM.

Phase 2:
When the user returns with the external answer:
- Identify convergences.
- Identify divergences.
- Explain what changes in the recommendation.
- Give the final recommendation.
